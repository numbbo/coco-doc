Detailed Description

Implementation of the bbob logger.

Logs the performance of a single-objective optimizer on noisy or noiseless problems. It produces four kinds of files:

  • The "info" files ...
  • The "dat" files ...
  • The "tdat" files ...
  • The "rdat" files ...
